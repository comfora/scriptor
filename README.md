# Scriptor

Scriptor is an open-source CLI tool designed for simple management of your local and cloud environments, tailored to your home labs or business environments.

Ranging from simple VPN or proxy installations to complex security configurations, Scriptor is designed to be a simple, easy-to-use tool for managing your environments from one installable Python package.
